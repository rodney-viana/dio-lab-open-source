Tags

Dentro do curso, ficou bem claro que o HTML trabalha com tags. Temos as que se fecham sozinhas e as que precisam ser fechadas manualmente. Também foi muito bem explicado que o uso das tags deve ser feito da forma mais semântica possível, pois é por meio da semântica do HTML que os leitores de tela irão fazer a leitura do conteúdo para pessoas com deficiência visual. Dentro deste assunto, vimos tags como <header>, <main> e <footer>, todas elas essenciais para um HTML semântico.
